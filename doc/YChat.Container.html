Class: Container

YChat. Container

new Container(parentElement, options)

Creates a new YChat.Container. YChat container manages the html for a chat window's content. Think of the YChat.Container as the base class of all the views of the plugin
Parameters:
Name Type Description
parentElement object The element to ehich this chat container should be appended. Can be null for containers that should be appended to the body DOM element.
options object Options for this container. These options overide the YChat.containerDefaults.

Available options:

  • title - The title of the container, for exampe the name of the person in this chat.
  • canClose - If the container should have close button.
  • initialState - The initial state of this container - maxinized or minimized.
